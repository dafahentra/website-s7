Sector Seven

Moka OAuth Setup

Halaman ini digunakan sekali saja untuk mendapatkan refresh_token baru. Simpan token di Netlify Environment Variables setelah selesai.

1
Authorize ke Moka
Klik tombol di bawah untuk membuka halaman otorisasi Moka. Login dengan akun merchant, lalu klik Allow. Kamu akan di-redirect kembali ke halaman ini dengan code di URL.
2
Tukar Code → Token
Setelah redirect, code di URL akan otomatis terisi. Klik Exchange untuk menukar ke access + refresh token.
3
Simpan ke Netlify
Copy nilai MOKA_REFRESH_TOKEN di bawah, lalu paste ke Netlify → Site settings → Environment variables. Setelah itu, trigger re-deploy.
✅ Berhasil! Salin kedua nilai di atas ke Netlify Environment Variables, lalu trigger Deploy → Clear cache and deploy site. Setelah deploy selesai, kamu bisa tutup halaman ini.
⚠️ Jangan share refresh token ini ke siapapun. Hapus halaman moka-setup.html dari public folder setelah selesai.
