Sector Seven — Moka Setup

OAuth Authorization

Jalankan halaman ini sekali untuk mendapatkan token akses Moka. Kamu akan diarahkan ke halaman Moka untuk memberikan izin.

Konfigurasi

Redirect URI https://sectorseven.space/callback
Scope profile library transaction checkout
Response Type authorization_code

Langkah Setup

  • 1 Klik tombol di bawah untuk membuka halaman authorize Moka.
  • 2 Login ke akun Moka kamu dan klik Allow.
  • 3 Kamu akan diarahkan ke halaman callback — token akan otomatis diproses oleh React.
  • 4 Salin MOKA_REFRESH_TOKEN yang muncul ke Netlify env vars, lalu redeploy.

Halaman ini hanya untuk setup awal.
Setelah MOKA_REFRESH_TOKEN tersimpan di Netlify,
halaman ini tidak perlu diakses lagi.
